무엇을 하나요

지금 재생 중

유튜브 뮤직, Spotify, 사파리 탭 — 시스템에 재생을 알리는 모든 앱. 앨범아트와 시크 슬라이더, 앨범 색을 따라가는 이퀄라이저.

싱크 가사

노래방처럼 지금 부르는 줄이 따라옵니다. 노치 아래 작은 가사 바로 띄워둘 수도 있어요.

다음 일정

노치를 펼치면 다음 일정이 한 줄로. 시작 10분 전엔 조용히 배너로 알려줍니다. 구글 캘린더는 맥 캘린더 연동으로.

볼륨 · 밝기 · 에어팟 · 충전

조절하는 순간 노치가 옆으로 부풀며 보여주고, 끝나면 스르륵 돌아갑니다.

Claude Code

답변이 끝나면 노치에서 클로드가 알려줘요. 클릭하면 그 창으로, 세션 잔량도 함께.

그리고

음악이 없을 땐 눈만 깜빡이며 기다립니다. 맥이 바쁠수록 눈도 바빠져요.

설치

  1. Cove.dmg를 받아 Applications로 드래그하세요.
  2. 첫 실행 때 경고가 뜹니다. 아직 애플 공증 서명이 없어서 그래요. 시스템 설정 → 개인정보 보호 및 보안 → "그래도 열기", 또는 터미널에서 xattr -cr /Applications/Cove.app
  3. 메뉴바의 노치 아이콘 → 설정에서 로그인 시 자동 시작을 켜두면 끝. 캘린더 권한은 일정 표시용이에요.

Claude Code와 함께

Stop 훅 한 줄이면 응답이 끝날 때마다 노치가 알려줍니다. 아무 스크립트에서나 Cove notify "메시지"도 됩니다.

{
  "hooks": {
    "Stop": [{ "hooks": [{
      "type": "command",
      "command": "/Applications/Cove.app/Contents/MacOS/Cove notify \"답변 완료\" 2>/dev/null || true"
    }] }]
  }
}
