노치 하나로 이만큼

지금 재생 중

유튜브 뮤직, Spotify, 사파리 — 시스템에 재생을 알리는 모든 앱을 지원해요. 앨범아트, 시크 슬라이더, 앨범색 이퀄라이저까지.

싱크 가사

노래방처럼 지금 부르는 줄이 따라와요. 노치 아래 플로팅 가사 바로 어디서든 흥얼거리세요.

다음 일정

노치를 펼치면 다음 일정이 보이고, 시작 10분 전엔 배너로 알려줘요. 구글 캘린더도 맥 캘린더 연동으로 OK.

다이나믹 HUD

볼륨 · 밝기 조절, 에어팟 연결, 충전기 연결 — 노치가 옆으로 부풀며 다이나믹 아일랜드처럼 보여줘요.

Claude Code 알림

Claude가 답변을 마치면 노치에 클로드가 빼꼼 알려줘요. 클릭하면 바로 그 창으로, 세션 잔량도 함께.

Cove 캐릭터

음악이 없을 땐 노치 옆에서 눈만 깜빡여요. 맥이 바쁠수록 눈도 바빠지는 건 비밀.

설치는 금방이에요

1

Cove.dmg 다운로드

위의 다운로드 버튼을 누르고, 열린 창에서 Cove를 Applications로 드래그하세요.

2

첫 실행 허용

아직 애플 공증 서명이 없어 경고가 떠요. 시스템 설정 → 개인정보 보호 및 보안 맨 아래 "그래도 열기"를 누르거나, 터미널에서 xattr -cr /Applications/Cove.app 한 줄이면 됩니다.

3

노치에 정박 완료

메뉴바의 노치 아이콘 → 설정에서 로그인 시 자동 시작을 켜두면 항상 함께해요. 캘린더 권한은 일정 표시용이에요 (선택).

Claude Code와 함께 쓰기

Stop 훅 한 줄이면 Claude가 응답을 마칠 때마다 노치가 알려줘요. 어떤 스크립트에서든 Cove notify "메시지"로 배너를 띄울 수 있어요.

{
  "hooks": {
    "Stop": [{ "hooks": [{
      "type": "command",
      "command": "/Applications/Cove.app/Contents/MacOS/Cove notify \"답변 완료\" 2>/dev/null || true"
    }] }]
  }
}
